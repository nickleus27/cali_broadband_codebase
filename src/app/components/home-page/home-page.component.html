California Broadband Study

Welcome to the CSUMB Calspeed broadband study (Calspeed). From this website, results of the California broadband study can be viewed as different graphs. The different graph options can be chosen from the top menu bar on the right hand side. Once all options are selected on the left hand sidebar menu, the Graph button will be enabled to display graph results. Also, the Comparison button can be selected to allow for options for a second graph (and third graph) in the sidebar menu.
The line graph lets you select a specific download speed range for a carrier and phone model. Each point on the graph will show the percentage of tests for the carrier/phone model that were within the chosen download speed range.
The options in the sidebar menu on the left let you choose graph results for county wide data. For example, you could select a carrier, phone model, and a county and see the average download speed recorded in that county as a line graph from different batches of tests going back to 2017. A note about the phone model option. If a batch of tests does not include that phone model, the next most current model will be chosen and displayed as a data point.
